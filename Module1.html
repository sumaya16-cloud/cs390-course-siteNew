Module 1: Foundations and Mathematical Readiness

Welcome to Module 1. This module introduces the mathematical language and basic concepts we will use throughout CS 390. Before we study finite automata, regular expressions, context-free grammars, and Turing machines, we need to become comfortable with sets, strings, alphabets, languages, functions, and proof-style reasoning.

What You Should Complete

  1. Read the Module 1 overview.
  2. Review the mathematical foundations notes.
  3. Complete the practice questions.
  4. Check Canvas for any required quiz or discussion.

Learning Objectives

  • Define alphabets, strings, and languages.
  • Use basic set notation correctly.
  • Explain why mathematical precision matters in theoretical computer science.
  • Prepare for deterministic finite automata in Module 2.
